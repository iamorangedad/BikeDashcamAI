一个端到端的 MVP 项目：“基于语义的智能视频搜索引擎”

1. 数据准备: 下载一个小型视频数据集（如 MSR-VTT 或 ActivityNet）。
2. 模型层: 使用 SigLIP 提取视频帧特征，使用 TimeSformer 提取时序特征，进行加权融合。
3. 索引层: 使用 Milvus 搭建向量库，构建 HNSW 索引。
4. 服务层: 将模型转化为 TensorRT 引擎，使用 Triton 部署。
5. 应用层: 编写一个简单的API，输入自然语言（例如：“一只狗在草地上接飞盘”），返回最匹配的视频片段及其精确的时间戳（例如：Video_03.mp4 的 00:15 - 00:20）。
6. 评估: 跑通 R@10 和 mAP 指标，并记录推理的平均延迟（Latency）。
